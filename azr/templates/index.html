Tokenomics

This token was created with the aim of creating a crypto around the game League of Legends so that it would depict the performance of an average player in the different game modes. The token is tied to a developer's game performance.

Shurima Influence

The token presents a preference for all those who are the champions of the game from Shurima. Obviously there is a main focus on the game history between Azir and Xerath. Those champions will give a boost to tokens that will be deleted.

EVERY DAY SOMETHING BURNS

And is not for the f*** ignite. each day, the result of the matches generates a number of tokens that will be removed to increase the value of AzirCoin. There are bonuses for wins and losses, also based on champions played (Shurima region).

No Game No Problem

If no new matches are found to be registered, the previous result will automatically be taken as the value for eliminating other tokens.

Constant Evolution

Depending on events, leagues and LoL evolutions, the burn value changes periodically, leading to ever-improving AZR value.

Concept

Azircoin was created with the goal of being a currency that can be used by the players themselves and to gain value over time. In addition, in subsequent developments the same users who buy azircoin and burn it will get in return another token that can be spent on a marketplace that is under construction, it will be possible to buy products from riot games and general gaming-related products from it.

Azir coin is based on technology An ERC20 token contract keeps track of fungible tokens.
